## Проект интернет-магазина по заказу пиццы - React Pizza

**Stack:**

- ReactJS + хуки
- React Router
- Redux
- Redux thunk
- json-server
- Axios
- classnames

## Запуск приложения
### `npm start`

## Запуск сервера для подгрузки пицц
### `npm run server`

**Оценить работу приложения можно здесь:** https://react-pizza-layout.herokuapp.com
